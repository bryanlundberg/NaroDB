# Installation

To install NaroDB, you can use any of the following package managers. Choose the one that best fits your development environment.

::: code-group

```bash [pnpm]
pnpm add @narodb/naro
```

```bash [npm]
npm install @narodb/naro
```

```bash [yarn]
yarn add @narodb/naro
```

```bash [bun]
bun add @narodb/naro
```

:::
